# GlucoGuard AI v2 🩺

**AI-Powered Diabetes Management System for the Indian Market**

Built for Dr. Nadeem Rais Partnership Demo - March 2026

---

## 🎯 Features

### 1. **Glucose Monitoring Dashboard**
- Real-time glucose trend charts (24h, 7d, 30d views)
- Time-in-Range visualization
- Average glucose tracking
- HbA1c estimation

### 2. **AI Meal Planner**
- Personalized meal recommendations based on glucose levels
- 40+ Indian foods database with glycemic index data
- Regional cuisine support (North, South, East, West Indian)
- Carb counting and portion guidance

### 3. **Predictive Alerts**
- Hypoglycemia/hyperglycemia risk prediction
- Pattern detection (Dawn phenomenon, post-meal spikes)
- Customizable alert thresholds
- Email notifications

### 4. **Indian Food Database**
- 40+ Indian foods with complete nutrition info
- Glycemic index classification
- Regional and category filters
- Hindi names for local accessibility

---

## 🚀 Tech Stack

- **Frontend:** React 18 + TypeScript
- **Styling:** Tailwind CSS
- **Charts:** Chart.js + react-chartjs-2
- **Icons:** Lucide React
- **Build Tool:** Vite

---

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/chatgptkrylor/glucoguard-v2.git
cd glucoguard-v2

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
